Win 7 中用 Putty 远程连接 Ubuntu 16

Posted on 2017-01-13 | Post modified 2017-01-13 | In Win-2-Ubuntu | | Visitors |

查看有没有安装SSH server

刚安装好Ubuntu 16.04系统之后，查看系统中是否已经安装SSH server，

dpkg -l | grep ssh

or,

ps -e | grep sshd

安装SSH server

sudo apt-get install ssh

如果此时安装不成功，通常需要更新系统，

sudo apt-get update

通常安装之后，保持默认设置即可。

win7上用putty连接

一般测试即可联通,

其它一些额外设置

启动服务器

/etc/init.d/ssh stop #停止
/etc/init.d/ssh start #启动
/etc/init.d/ssh restart #重启

SSH配置

修改配置文件，

sudo gedit /etc/ssh/sshd_config

并重启服务，

/etc/init.d/ssh restart
