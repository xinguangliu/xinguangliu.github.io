工作记录：在Ubuntu16.04安装配置MySQL

安装

在线安装

sudo apt-get install mysql-server

安装后的目录地址: /etc/mysql/

  • 检查MySQL服务器系统进程:
    ps -aux|grep mysql
  • 检查MySQL服务器占用端口
    netstat -nlt|grep 3306
  • 通过启动命令检查MySQL服务器状态
    sudo /etc/init.d/mysql status
  • 通过系统服务命令检查MySQL服务器状态
    service mysql status

登录

连接MySQL格式：

mysql -h 主机地址 -u用户名 -p用户密码

连接本机数据库：

mysql -uroot -p

重置root账户密码：

GRANT ALL PRIVILEGES ON *.* TO root@localhost IDENTIFIED BY "myPassWord"

退出：exit;

远程登录

在Ubuntu下MySQL缺省是只允许本地访问的，远程登录需要设置两个地方：

  1. 命令授权
    GRANT ALL PRIVILEGES ON *.* TO <user>@"%" IDENTIFIED BY '<password>' WITH GRANT OPTION;
    FLUSH PRIVILEGES;

  2. 修改配置文件
    sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf
    //找到如下内容，并注释
    bind-address = 127.0.0.1

之后重启MySQL服务：sudo service mysql restart

改变数据存储位置

默认存储位置在 sudo gedit /etc/mysql/mysql.conf.d/mysqld.cnf 中的 datadir= /var/lib/mysql, 通过Linux系统的软连(ln -s)接来做。当我们新挂载一块硬盘，停止MySQL服务，然后把/var/lib/mysql目录移动到新的硬盘存储，在/var/lib/mysql处建立指定新位置的软连接就行了。
